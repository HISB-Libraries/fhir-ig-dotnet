# fhir-ig-odh-dotnet

A .NET Library for ODH FHIR IG (https://build.fhir.org/ig/HL7/us-odh/)

This library contains .net extensions and value sets for FHIR IGs. This library is based on .NET FHIR classes (https://github.com/FirelyTeam/firely-net-sdk).

Sources are available in https://github.com/HISB-Libraries/fhir-ig-dotnet
